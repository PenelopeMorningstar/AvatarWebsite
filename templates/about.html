Welcome to our Robotics Project repository! This project showcases three exciting autonomous robots we designed and implemented: Line Follower Bot, Line Maze Bot, and Physical Maze Bot. Each bot demonstrates different aspects of robotics, sensor integration.

ğŸš€ Projects Overview

1. Line Follower

  • Objective: Enable the robot to follow a predefined line path automatically.
  • Features:
    • Uses IR sensors for line detection.
    • PID control for smooth navigation.
    • Adjustable speed and sensitivity.
  • Applications: Educational robotics, autonomous navigation.

2. Maze Follower

  • Objective: Solve a maze using autonomous navigation algorithms.
  • Features:
    • Distance sensors for obstacle detection.
    • Implements right-hand rule maze solving.

3. Physical Maze Bot

  • Objective: Navigate a real-life physical maze.
  • Features:
    • Hybrid line following + obstacle detection.
    • Real-time decision making.
  • Applications: Real-world robotic problem-solving demos.
